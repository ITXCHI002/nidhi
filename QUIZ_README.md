# Basic G.K. Quiz Project

This project aims to create a basic General Knowledge (G.K.) quiz application. It is designed to test the user's knowledge across various subjects and topics.

## Features

- Multiple-choice questions
- Instant feedback on answers
- Scoring system to track progress

## Technologies Used

- HTML/CSS for layout and styling
- JavaScript for interactivity

## Installation

To run this project locally, clone the repository and open the `index.html` file in your web browser.
